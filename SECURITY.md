# Security Policy

## Supported Versions

Security fixes target the latest published version of VueGridle.

## Reporting a Vulnerability

Please do not open a public issue for a security vulnerability.

Report privately by email to `artemdoroshko555@gmail.com` or through GitHub Security Advisories when available.

Include:

- Affected version
- Reproduction steps
- Impact
- Suggested fix, if known

I will acknowledge valid reports as soon as possible and coordinate a fix before public disclosure.
